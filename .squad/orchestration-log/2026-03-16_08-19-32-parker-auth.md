## What Was Built

SQLite-based local authentication system with:
- **Hashing:** Argon2id password hashing (configurable work factor)
- **Tokens:** JWT-based session tokens with configurable expiration
- **Persistence:** SQLite user table with email/password/created_at
- **HTTP Interface:** `/auth/login`, `/auth/logout`, `/auth/session` endpoints
- **Storage:** Cookie-based token delivery (`httponly`, `secure`, `samesite`)

## Implementation Details

- **Test coverage:** 129 unit + integration tests (pytest suite)
- **Baseline security:** Argon2id + JWT per OWASP guidelines
- **Error handling:** Timing-safe comparisons, proper password reset flows
- **Logging:** Structured auth events with PII scrubbing
- **Dependencies:** No external auth framework — purely stdlib + cryptography
