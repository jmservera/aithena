### Ash — Solr 10 Language-Models Module Deep Research
- **Question:** Can Solr 10's language-models module replace our embeddings-server?
- **Research Scope:** Module capabilities, remote vs. local inference, E5 model compatibility, performance
- **Verdict:** ❌ Cannot replace embeddings-server
- **Key Findings:**
  - Remote API only (OpenAI, Cohere, HuggingFace, MistralAI) — no local inference
  - No E5 prefix preprocessing ("query: " / "passage: ") support
  - One-by-one encoding (vs. our 50-doc batches) with performance warnings
  - No GPU support; in-process ONNX tracked but not implemented (SOLR-17446)
- **Decision:** Keep current embeddings-server, monitor SOLR-17446 for re-evaluation
- **Report:** `docs/research/solr10-language-models-embeddings.md` (5 KB)
