## User Directive

**Directive:** Always branch from `dev`, never from `main`

This directive supersedes any prior assumptions about branching from main. All future feature work, bug fixes, and releases must branch from dev:

```bash
# ✅ CORRECT
git checkout -b feature/xxx origin/dev

# ❌ INCORRECT
git checkout -b feature/xxx origin/main
```

Only release PRs go from dev → main (via `gh pr create --base main --head dev`).
