**Status:** MERGED  
**Issue:** #887 (admin login loop)

- JWT session with `role == "admin"` (browser access via React SPA)

**Impact:**
- Admin dashboard now fully functional without X-API-Key
- Scripts/CI workflows unchanged (API-key flow still supported)
- All team members: when adding auth gates, test both paths
