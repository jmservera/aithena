### 2026-03-14T15:12: User feature request — PDF page navigation from search results
**By:** jmservera / Juanma (via Copilot)
**What:** Search results must show which page number(s) contain the matching text. Clicking a result should open the PDF at the correct page. This requires: (1) Solr to track page numbers during indexing, (2) search API to return page numbers with highlights, (3) PDF viewer to open at a specific page.
**Why:** Critical usability — users need to find the exact location of search hits in large PDFs
