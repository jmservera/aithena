# What We're Focused On

Building a book library search engine powered by Apache Solr. The system indexes PDFs from `/home/jmservera/booklibrary`, extracts metadata (author, date, language) from filenames/folders/content, supports multilingual full-text search (Spanish, Catalan, French, English), and provides a React UI for searching, filtering, and viewing PDFs. Future enhancement: local multilingual embedding model for semantic search alongside Solr's native keyword search.
