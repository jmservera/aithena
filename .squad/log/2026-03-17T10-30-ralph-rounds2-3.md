### Detailed Findings

**#404 — Stats Count Discrepancy (Ash)**

**Problem:** `/v1/stats` returns 127 total books when only 3 books exist
- Root cause: Solr contains 127 documents (3 parent books + 124 child chunks)
- Stats query counts all documents, not unique books

**Proposals:**
1. **Phase 1 (Quick Win v0.8.0):** Use parent_id_s grouping, no schema changes (10-line change)
2. **Phase 2 (Full Solution v0.9.0):** Add doc_type_s discriminator field for proper parent/child separation

**Architectural Review:** Awaiting Ripley approval before implementation
