## Verified Scope for v1.14.1

The clearest `v1.14.x` operator-facing workflows verified in the current tree are:

- **Backup dashboard and restore wizard** in the React UI at `/admin/backups`
- **Protected full reindex endpoint** at `POST /v1/admin/reindex`
- **Admin reindex page** in the embedded admin app for destructive full-library rebuilds

These workflows are now documented more explicitly in the manuals shipped with this release.

### What stayed stable

No user-facing regressions or breaking changes can be inferred from the supplied changelog, PR export, or CI export. The existing end-user workflows already documented in the user manual remain the current source of truth for:

- login and account management
- search modes and diacritic-insensitive search
- PDF viewing and download actions
- collections and book-detail workflows
