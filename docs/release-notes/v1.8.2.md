# Release Notes — v1.8.2

**Release date:** 2026-03-19
**Milestone:** [v1.8.2](https://github.com/jmservera/aithena/milestone/25?closed=1)

## Summary

Infrastructure improvements: fix file upload size limit, remove legacy Streamlit admin, add infrastructure management links to the React admin page, and fix Dependabot CI.

## Fixed

- **File upload failing with size error** — added `client_max_body_size 64m` to nginx config. The reverse proxy was rejecting uploads before they reached the backend's 50MB limit. Frontend error message improved to be generic rather than hardcoding the limit. (#596)
- **Dependabot automerge CI** — the workflow's test step used file-based detection (`pyproject.toml`) to choose between uv and pip, but embeddings-server now has `pyproject.toml` while still using pip. Changed to service-name-based detection. (#598)

## Removed

- **Streamlit admin service** — the `streamlit-admin` Docker Compose service, nginx proxy blocks, and backend container health checks have been removed. The React admin page (`/admin/`) now provides full feature parity for document management, queue monitoring, and system status. A 301 redirect from `/admin/streamlit` to `/admin/` ensures a graceful transition for existing bookmarks. (#587)

## Added

- **Infrastructure UI links** — the React admin page now includes card-style links to Solr Admin (`/admin/solr/`) and RabbitMQ Management (`/admin/rabbitmq/`) with i18n support in all four languages (en, es, ca, fr). (#588)

## Upgrade Notes

- If you have bookmarks or scripts pointing to `/admin/streamlit/`, they will automatically redirect to `/admin/`.
- The `streamlit-admin` container will no longer start. You can safely remove any Streamlit-specific environment variables from your `.env` file.
- The `src/admin/` source directory is preserved for reference but is no longer deployed.
