# Aithena v1.9.1 Release Notes — Docker Build Fix

_Date:_ 2026-03-20  
_Prepared by:_ Squad (Ralph — Team Lead)

## Summary

Patch release that fixes the aithena-ui Docker build failure caused by a missing `.npmrc` file in the Dockerfile build context.

## Issues Resolved

| # | Title | Type |
|---|-------|------|
| [#616](https://github.com/jmservera/aithena/issues/616) | aithena-ui Docker build fails — .npmrc not copied before npm ci | Bug |

## Changes

### Fixed

- **aithena-ui Dockerfile**: Copy `.npmrc` alongside `package*.json` before running `npm ci` so that the `legacy-peer-deps=true` setting is respected. Without this, `npm ci` fails with an ERESOLVE error because `eslint-plugin-jsx-a11y@6.10.2` has a peer dependency on `eslint ^3-^9` while the project uses `eslint@10.0.3`.

## Upgrade Notes

No configuration changes required. Rebuild the aithena-ui Docker image to pick up the fix.
